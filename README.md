# Rental Yield Token (RYT)

A real ERC-20 smart contract that tokenises the income stream of a UK residential buy-to-let property. Built as part of the Individual Coursework for Decentralised Finance and Blockchain (IFTE0007) at University College London.

---

## What This Project Does

This contract issues 1,000,000 RYT tokens on the Ethereum blockchain. Each token represents a fractional claim on the monthly rental income generated by a residential property held inside a Special Purpose Vehicle (SPV). Token holders receive proportional income distributions automatically via smart contract, without the need for any financial intermediary.

Rather than tokenising the physical building itself, this scheme tokenises the right to receive rental income. Think of it like holding shares in a company that owns the property rather than owning the property directly. Token holders do not own the bricks and mortar but they receive a slice of what the property produces every month.

## How It Works

1. Tenant pays monthly rent into the SPV bank account
2. SPV operator converts net income into USDC stablecoin
3. Chainlink oracle confirms the deposit on chain
4. Smart contract automatically distributes USDC proportionally to all token holders
5. Every distribution is permanently recorded and publicly verifiable on Etherscan

## Smart Contract Functions

### distributeIncome(uint256 totalIncome)
Called by the contract owner each month when rental income is received. Emits an IncomeDistributed event on chain confirming the distribution amount and timestamp.

### redeem(uint256 amount)
Called by token holders after the 12 month lock-up period ends. Burns the specified number of tokens and emits a TokensRedeemed event. Investors use this to exit their position.

### burn(uint256 amount)
Allows token holders to voluntarily destroy their tokens at any time. Reduces circulating supply and emits a TokensBurned event.

### isLockUpEnded()
View function. Returns true if the 12 month lock-up period has ended, false if it is still active.

### lockUpTimeRemaining()
View function. Returns the number of seconds remaining in the lock-up period. Returns 0 if lock-up has ended.

### getHolderBalance(address holder)
View function. Returns the RYT token balance of any wallet address.

## Why Tokenise Rental Income

The UK private rental sector covers approximately 4.6 million households yet investing in it has always required a deposit of between £50,000 and £100,000, excluding the vast majority of ordinary investors. Tokenisation breaks the income stream into small affordable units that anyone can buy into from as little as £50.

Key problems this scheme solves:

- High capital barriers — minimum investment reduced from £50,000 to £50
- Illiquidity — tokens tradeable on decentralised exchanges after lock-up
- Lack of transparency — all distributions publicly verifiable on chain
- Slow settlement — token transfers settle in seconds not months
- Geographic barriers — any global investor with a crypto wallet can participate
